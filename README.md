# NTU Women in Tech

Official website for NTU Women in Tech, building the next generation of tech leaders through flagship events, mentorship, and community.

## Getting Started

Install dependencies with npm:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:8080`

## Build

```bash
npm run build
```

## Technology Stack

- **React** + **TypeScript** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Shadcn UI** - Component library
- **React Router** - Navigation
- **Three.js** - 3D graphics
